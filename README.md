# Interactive Navigation Menu



### Task Objective
To create an interactive navigation menu that changes style on scroll and hover. The navigation bar remains fixed and visible on all pages.

---

##  Technologies Used
- HTML
- CSS
- JavaScript

---

##  Features
- Fixed navigation bar
- Scroll-based background color change
- Hover effects on menu items
- Clean and structured multi-section layout

---

##  Knowledge Gained
- DOM manipulation using JavaScript
- Scroll event handling
- CSS hover styling
- Improved layout structuring
- Better understanding of responsive design basics
